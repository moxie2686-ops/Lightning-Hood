# Prompt: AI Crypto Analytics Platform

## 1. Konsep & Gaya Visual
Buat website modern bertema **AI Crypto Analytics Platform**. Gunakan logo yang disediakan sebagai identitas utama dan jadikan palet warnanya sebagai inspirasi desain keseluruhan.

- Gaya: futuristik, premium, dan Web3
- Efek: glassmorphism, soft glow, animasi halus (micro-interactions)
- Layout: minimalis, banyak whitespace, grid rapi
- Kesan akhir: profesional & setara platform enterprise seperti Arkham Intelligence, Nansen, DexScreener, GeckoTerminal, Dune, Uniswap, Linear, dan Vercel

## 2. Tech Stack
- Next.js + React + TypeScript
- Tailwind CSS
- Framer Motion (animasi)
- Responsif penuh (mobile, tablet, desktop)
- SEO-friendly (meta tags, semantic HTML, OG image)
- Performa tinggi (lazy load, optimized images, minim layout shift)

Nama proyek, logo, warna, dan branding dibuat sebagai **placeholder** agar mudah diganti sendiri nantinya.

## 3. Deskripsi Platform

**Introduction:**
Platform AI Crypto Analytics yang menyediakan analisis pasar kripto secara real-time melalui data on-chain, AI, dan blockchain intelligence. Membantu pengguna melacak token, memantau aktivitas whale, menganalisis sentimen pasar, mengelola portofolio, serta melakukan swap aset digital dengan pengalaman cepat, aman, dan intuitif.

**How It Works:**
Platform mengumpulkan data dari berbagai blockchain dan protokol DeFi, lalu AI menganalisis harga, volume, likuiditas, transaksi wallet, smart money, dan sentimen pasar untuk menghasilkan insight, skor risiko, serta notifikasi real-time. Pengguna dapat menghubungkan wallet, mencari token, melakukan swap, memantau portofolio, dan mengakses dashboard analitik dalam satu platform.
